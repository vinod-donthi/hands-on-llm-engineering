## How to use this

1. **While reading theory** — if you see an unfamiliar term (e.g. XGBoost), check the [Week 1 glossary](../week-01/resources/glossary.md) for a one-line definition.
2. **When curious** — follow the glossary link to the relevant appendix page for a 5–10 minute read.
3. **When interviewing** — revisit appendix pages tied to system design questions (fraud detection, tabular vs text, model choice).
